Technical Documentation

System Architecture.

A multi-layered system designed for robust data acquisition, processing, intelligent mapping, and intuitive user presentation.

01

Data Collection Layer

Scheduled data-mining scripts autonomously extract public internship listings from targeted university portals, career platforms, and company web pages, mimicking precise human interactions.

02

Text Extraction Layer

Utilizes robust OCR (Optical Character Recognition) technologies like Tesseract and dynamic layout interpreters to extract structured text directly from image-based job ads and complex DOM nodes.

03

Data Processing Layer

Normalizes extracted fields, filters irrelevant context, dynamically patches missing gaps via Natural Language Processing, and handles strict duplicate detection inside the central relational database.

04

Matching Layer

Analyzes student profiles, resumes, and predefined job criterias against real-time normalized listings using term frequency weighting systems and skill map algorithms to discover best-fit opportunities.

05

Presentation Layer

A performant Streamlit-powered dashboard bridging our dataset to students, highlighting customized insights, extensive filtering, and rapid redirections back to application sources.
